
# Health Insurance Costs 




## Describe Project

Multiple linear regression model for medical cost dataset to predict medical costs charged by health insurance. The dataset consists of age, gender, BMI (body mass index), children, smoker, and region.
## Objectives

Forecast medical costs for health insurance.
